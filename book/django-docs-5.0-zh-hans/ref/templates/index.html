模板¶

Django 的模板引擎提供了一个强大的迷你语言来定义你的应用程序的面向用户层，鼓励应用程序和表现逻辑的清晰分离。任何懂 HTML 的人都可以维护模板，不需要 Python 知识。关于入门材料，请参见 模板 主题指南。
