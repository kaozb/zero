Django 3.0.11 版本发行说明¶

2020 年 11 月 2 日

Django 3.0.11 修复了 3.0.7 版本中的回归问题，并添加了与 Python 3.9 的兼容性。

漏洞修复¶

  • 修正了 Django 3.0.7 中一个在 GROUP BY 子句中没有使用 Subquery() 别名的缺陷（#32152）。
