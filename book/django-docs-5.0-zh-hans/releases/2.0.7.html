Django 2.0.7 版本发行说明¶

2018 年 7 月 2 日

Django 2.0.7 修复了 2.0.6 版本中的一些错误。

漏洞修复¶

  • 已修复管理员 changelist 在页面排序中使用没有 asc() 或 desc() 的查询表达式时的崩溃问题（#29428）。
  • 已修复在 ModelAdmin.ordering 中使用查询表达式时，导致管理员检查崩溃的问题（#29428）。
  • 已修复与 MySQL 8 中的 __regex 和 __iregex 查找相关的问题（#29451）。
  • 已修复在 Python 3.7 上使用命名空间包时导致迁移崩溃的问题（#28814）。
