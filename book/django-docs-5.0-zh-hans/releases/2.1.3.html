Django 2.1.3 版本发行说明¶

2018 年 11 月 1 日

Django 2.1.3 修复了 2.1.2 版本中的一些错误。

漏洞修复¶

  • 修复了 Django 2.0 中的一个回归问题，即将 Q 对象与 __in 查找和列表组合时会崩溃 (#29838)。
  • 修复了 Django 1.11 中的一个回归问题，即 django-admin shell 在启动时可能会挂起 (#29774)。
  • 修复了 Django 2.0 中的一个回归问题，在 MySQL 上使用 manage.py test --keepdb 时不再重用测试数据库 (#29827)。
  • 修复了一个回归问题，其中在 Model.save() 中错误地清除了使用 to_field 的缓存外键 (#29896)。
  • 修复了 Django 2.0 中的一个回归问题，即如果并发保存尝试创建相同的目录，FileSystemStorage 会导致 FileExistsError 错误 (#29890)。
